# block-youtube-shorts: Block YouTube Shorts (Desktop & Mobile) 

A simple Adblock/uBlock/Vivaldi filter list to completely block and hide YouTube Shorts on both desktop and mobile web versions.

## How to use:
Copy the link below and add it as a custom filter list in your adblocker or Vivaldi browser settings:

https://raw.githubusercontent.com/lukas1822/block-youtube-shorts/refs/heads/main/blockyoutubeshorts.txt

## What it does:
- Blocks direct access to /shorts/ URLs
- Hides the Shorts tab in the sidebar and navigation
- Removes Shorts carousels from the homepage and search results
